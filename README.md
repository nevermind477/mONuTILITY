## Описание

**mONuTILITY** — это консольный монитор системы (TUI), написанный на Python с использованием `curses` и `psutil`.

Показывает:

```
┌───────────────────────────────┐
│ CPU        ███████░░░  73%     │
│ RAM        █████░░░░░  42%     │
│ NET ↑↓     1.2MB/s            │
│ DISK       ████████░  81%     │
└───────────────────────────────┘
```

---

## Возможности

```
[✓] Мониторинг CPU (общий + по ядрам)
[✓] RAM и SWAP
[✓] Диски и использование
[✓] Сеть (скорость и трафик)
[✓] Температуры (если доступны)
[✓] Топ процессов
[✓] История нагрузки
```

---

## Управление

```
Tab        → переключение вкладок
1–5        → выбор вкладки
q / Esc    → выход
```


---

## Зависимости

```
psutil
curses (Linux/macOS)
windows-curses (Windows)
```

---

##  Минимальные требования

```
Терминал: ≥ 80x24
Python: 3.8+
```

---

##  Важно

```
- Это TUI приложение (НЕ GUI)
- Требует нормальный терминал
- В некоторых системах нет температуры
```
